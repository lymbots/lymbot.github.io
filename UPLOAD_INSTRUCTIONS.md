# Upload-instruktion

Denne mappe er klar til at blive uploadet direkte til roden af GitHub-repoet.

Repoets rod skal efter upload indeholde:

- `docs/`
- `manual_edit/`
- `CONTENT_GUIDE.md`
- `LIVE_GUIDE.md`
- `projektbrief_de_politiske_principper.md`
- `README.md`
- `UPLOAD_INSTRUCTIONS.md`

Vigtigt: upload indholdet af denne mappe direkte. Upload ikke en ekstra overmappe.

GitHub Pages skal stå til:

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/docs`

Siden ligger derefter på GitHub Pages-URL'en.
